# كوزات أونلاين

منصة اختبارات إلكترونية عربية للطلاب والدكاترة.

## المميزات

- تسجيل طالب بالاسم الرباعي والبريد والقسم ونوع الدراسة
- موافقة الدكتور على الحساب مع إشعارات داخل الموقع
- إنشاء اختبارات بأسئلة من 4 اختيارات ومدة بالدقيقة/الساعة
- تعيين الطلاب المسموح لهم بأداء كل اختبار
- وضع ملء الشاشة أثناء الامتحان مع تسجيل مخالفات مغادرة الصفحة
- حفظ تلقائي للإجابات وتسليم تلقائي عند انتهاء الوقت أو تجاوز المخالفات
- لوحة نتائج ومخالفات للدكتور

## التشغيل

### 1) تشغيل PostgreSQL
الأسهل عبر Docker:

```bash
docker compose up -d
```

أو ضع رابط PostgreSQL جاهز في ملف `.env` (Neon / Supabase / سيرفرك).

### 2) تثبيت وتشغيل الموقع

```bash
npm install
npm run db:setup
npm run dev
```

افتح: http://localhost:3000
